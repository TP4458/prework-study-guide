HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The Meta element defines information about data in the HTML document
  • The title element adds a title to the page, it should describe content and meaning of the page. Very importand for SEO optimisation.
  • The header element normally presents introductory content to the page
  • The headings are titles/subtitles diaplayed on the page. defined by h1 to h6 tags. must be used in order. Only one h1 tag should be used, for the main header.
  • The lists are created using ol and ul tags for ordered and unordered lists respectively. li tag is used for adding points to the list.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Selector defines rules and attributes to which declarations will apply
  • Declaration Consist of a property and assigned value

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: Adds all modifications to the staging area of current branch
  • git commit -m "message": commits changes to the branch, with a descriptive message
  • git pull origin main: pulls down any changes from the main branch in GitHub. Syncs with local branch. Always do this before pushing changes to assure latest ersion of the code is used.
  • git push origin branch-name: Pushes changes to the remote branch

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A function is a block of code designed to perform a particular task. It is exectued when ivoked(called)
  • Loops are used to run code multiple times, with diffrent values and/or under different conditions
